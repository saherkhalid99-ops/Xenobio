# XenoBio — Extremophile × Space Environment Matching Tool

A free, open-access decision-support tool for astrobiologists, planetary scientists, and mission planners.

## What It Does
XenoBio matches Earth's extremophile bacteria to space environments (Mars surface, Mars subsurface, Lunar surface, ISS) based on published survival and functional traits.

## Features
- 30 extremophile strains (natural + engineered)
- 8 space environments with confirmed NASA mission parameters
- Two modes: Explorer Mode and Researcher Mode
- Evidence quality ratings (High / Medium) per strain
- Earth analog environments for each strain
- 50 verified peer-reviewed references

## Live Tool
https://saherkhalid99-ops.github.io/Xenobio/xenobio-v1.html

## Scientific Basis
Built from 50 verified peer-reviewed references including Astrobiology, Nature Biotechnology, Science, Applied and Environmental Microbiology, Nature Reviews Microbiology, and NASA technical reports.
